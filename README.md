Quantum Interface Portfolio

This project is a custom personal website built using PHP, CSS, and JavaScript, designed with a clean structure and an interactive, modern interface. It features animated backgrounds, smooth transitions, a terminal-style loading screen, and dynamic elements that respond to user interaction.

The aim is to deliver a fast, lightweight, and visually distinctive experience using core web technologies without relying on heavy frameworks.

Features

Interactive UI with layered animations

Canvas-based background effects

Custom mouse-tracking behavior

Terminal-style preloader

Responsive layout

Slide-in WhatsApp contact drawer

Tech Stack

PHP

HTML5 / CSS3 / SCSS

JavaScript (ES6)

Canvas API

Project Structure
root/
├── index.php
├── views/
├── controllers/
├── assets/
│   ├── css/
│   ├── js/
│   └── media/
└── includes/

Setup

Upload the files to a PHP-enabled server.

Recommended PHP version: 7.4+

Open the project in your browser:

http://localhost/project-folder/

Notes

Animations are optimized for modern browsers.

Scripts are written in a modular and extendable structure.

You can adjust animation settings through the JS files.

License

This project is intended for personal use.
